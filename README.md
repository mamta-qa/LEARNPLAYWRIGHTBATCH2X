# Learn Playwright Batch 2x

This repository contains JavaScript practice exercises and notes for learning Playwright and JavaScript fundamentals.

## Structure

- `chapter_01_Basics/` — basic JavaScript setup and introductory examples
- `chapter_02_Javascript_Concepts/` — JavaScript concepts such as identifiers, naming rules, and comments
- `chapter_03_identifiers_litreals/` — additional exercises on identifiers and literals

## Contents

- `VS_shortcute.md` — Visual Studio Code shortcuts reference
- `chapter_02_Javascript_Concepts/06_identifiers_rules` — identifier rules examples
- `chapter_02_Javascript_Concepts/07_java_identifier_Part2.js` — naming style examples
- `chapter_02_Javascript_Concepts/08_comments.js` — comment syntax examples

## Usage

Open this repository in VS Code and run JavaScript files in a terminal or using a Node.js runtime.

## Notes

- JavaScript identifiers must start with a letter, underscore (`_`), or dollar sign (`$`).
- Multi-line comments use `/* ... */`.
- Use `camelCase` for variables and functions, and `PascalCase` for classes.
