# Visual Studio Code Shortcuts

## General Editing

- `Ctrl + S` : Save file
- `Ctrl + X` : Cut line (or selection)
- `Ctrl + C` : Copy line (or selection)
- `Ctrl + V` : Paste
- `Ctrl + Z` : Undo
- `Ctrl + Y` : Redo
- `Ctrl + D` : Select next occurrence
- `Ctrl + F` : Find
- `Ctrl + H` : Replace
- `Alt + Up/Down` : Move line up/down
- `Ctrl + /` : Toggle line comment
- `Shift + Alt + A` : Toggle block comment
- `Ctrl + Space` : Trigger suggestion / IntelliSense

## Navigation

- `                        Ctrl + P` : Quick Open file
- `Ctrl + Shift + O` : Go to symbol in file
- `Ctrl + G` : Go to line
- `Ctrl + T` : Go to symbol across workspace
-                    `F12` : Go to definition
- `Alt + F12` : Peek definition
- `Ctrl + -` : Navigate back
- `Ctrl + Shift + -` : Navigate forward
- `Ctrl + B` : Toggle sidebar visibility
- `Ctrl + \` : Split editor
- `Ctrl + 1`, `Ctrl + 2`, `Ctrl + 3` : Focus editor group

## Search and Replace

- `Ctrl + Shift + F` : Search in files
- `Ctrl + Shift + H` : Replace in files
- `F3` / `Shift + F3` : Find next / previous
- `Alt + Enter` : Select all occurrences of Find match

## Terminal

- `Ctrl + `` ` : Toggle integrated terminal
- `Ctrl + Shift + `` ` : Create new terminal
- `Ctrl + C` : Copy selected text in terminal
- `Ctrl + V` : Paste into terminal
- `Ctrl + Shift + C` : Open new external terminal

## Debugging

- `F5` : Start / continue debugging
- `F9` : Toggle breakpoint
- `F10` : Step over
- `F11` : Step into
- `Shift + F11` : Step out
- `Ctrl + Shift + F5` : Restart debugging

## File and Window Management

- `Ctrl + N` : New file
- `Ctrl + O` : Open file
- `Ctrl + W` : Close editor
- `Ctrl + K Ctrl + W` : Close all editors
- `Ctrl + Shift + N` : New window
- `Ctrl + Shift + W` : Close window

## Useful Commands

- `Ctrl + Shift + P` : Show Command Palette
- `Ctrl + K Z` : Zen Mode
- `Ctrl + K S` : Keyboard shortcuts reference
- `Ctrl + K Ctrl + S` : Open Keyboard Shortcuts editor
- `Ctrl + ,` : Open Settings

> Note: On macOS, replace `Ctrl` with `Cmd` and `Alt` with `Option`.
